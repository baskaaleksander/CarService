### Opis ról

- **Administrator** - pełny dostęp do systemu, zarządzanie użytkownikami, usługami, częściami i raportami
- **Mechanik** - obsługa zleceń serwisowych, dodawanie notatek diagnostycznych, zarządzanie częściami w zleceniach
- **Klient** - rejestracja pojazdów, składanie zleceń serwisowych, przeglądanie historii i dodawanie opinii
